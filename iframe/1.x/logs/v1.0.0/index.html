Admui-iframe v1.0.0 正式版发布

2017/12/25

Admui-iframe 通用管理系统快速开发框架是一个基于最新 Web 技术的企业级通用管理系统快速开发框架，可以帮助企业极大的提高工作效率，节省开发成本，提升品牌形象。您可以 Admui 为基础，快速开发各种MIS系统，如CMS、OA、CRM、ERP、POS等。Admui 紧贴业务特性，涵盖了大量的常用组件和基础功能，最大程度上帮助企业节省时间成本和费用开支。

根据用户的需求，我们推出了 Admui 的 iframe 版本 Admui-iframe，即：使用 iframe 实现标签页功能，切换标签页不会丢失页面操作状态，不过度依赖后端服务器环境，双击即可打开静态页面的版本（部分含ajax内容的页面除外）。

Admui-iframe v1.0.0 版本包含：

UI 示例

  • UI 组件(布局、基本、进阶、组件、小工具)
  • 页面示例(首页、常用、团队)
  • 表格(基本表格、树形表格、DataTables)
  • 表单(基本、质感、高级、布局、向导、验证、预设、编辑、编辑器、图片裁剪、文件上传)
  • 图表(简单图表、ECharts)
  • 系统(菜单、用户、日志、黑名单、系统、账户)

共六大模块，涵盖1000+ 基础组件，200+ UI示例页面，50+ 完整示例页面。

基础功能

  • 登录
  • 菜单管理
  • 用户管理
  • 日志信息
  • 黑名单
  • 系统设置(显示设置、日志设置)
  • 账户信息(消息、日志、密码、显示)
  • 消息中心(Websocket实时推送)
  • 错误页面(维护中、无权限、404等)
  • 网站地图
