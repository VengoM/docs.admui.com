概述

Admui是一款管理系统快速开发框架，采用 spring boot 2.0.4、spring5.0.8、SpringMvc、mybatis、shiro、swagger、redis、quartz、thymeleaf 等技术开发；现已实现的功能模块：菜单管理，组织机构、用户管理，角色管理，权限管理，日志设置，显示设置，账户信息，日志信息，黑名单等。

环境支持

Admui 可以很好的支持 JDK1.7 及以上版本，MySQL5.5 及以上版本，操作系统不限。

开源框架

Admui 使用了Spring MVC + MyBatis作为基础开发框架：

  • Spring Boot 2.0.4.RELEASE
  • Spring 5.0.8. RELEASE
  • MyBatis 3.4.6
  • Shiro 1.4.0
  • poi 4.0.0
  • Spring Boot 自带的 quartz，websocket 等
