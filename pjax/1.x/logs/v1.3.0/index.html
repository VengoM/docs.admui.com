Admui-pjax v1.3.0 更新日志

2018/03/10

Admui-pjax v1.3.0 重点增加了标签页的浏览器存储功能，对核心方法做了简单的调整，同时优化和修复若干问题，v1.3.0 更新日志如下：

新增

  • 标签页的浏览器存储功能，记录访问的内容页；
  • bootstrap中tooltip和popover控件的data-API调用注册文件；

优化

  • 替换部分页面中错误的单词拼写和无效的url地址；
  • 简单修改核心代码，提高代码可读性；
  • 修改pjax请求触发条件及url地址定位当前左侧菜单；
  • 修改标签页创建功能；
  • 修改用户管理页面新增和编辑功能；
  • 修改表单验证示例页面英文提示信息；
  • 修改博客示例页面瀑布流插件初始化调用；
  • 简单修改jquery.pjax.js代码，提高代码可读性；
  • 删除已废弃控件样式；
  • 删除SystemController及MenuServiceImpl中的无用代码；

修复

  • 修复显示设置改变导航条颜色出现与选中颜色不同问题；
  • 修复菜单管理页面顶部菜单名称修改时出现的数据字符串转Json对象问题；
  • 修复高级表单示例页面个别控件示例未生效问题；
  • 修复dataTables表格示例页面左侧菜单展开收起问题及个别表格底部错位问题；
  • 修复统计图示例页面图表定时器在离开页面时清除问题；
  • 修复删除submenu时未刷新导致绑定数据一直存在，切换菜单后删除元素依然存在；
  • 使用ThreadLocal处理request response session loginUser，避免在高并发情况下可能会发生的线程不安全问题；
